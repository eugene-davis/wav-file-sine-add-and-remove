To run phase 1 of the CPE Project, use the command:
381-project-phase-1 <input_wav> <output_wav>

For example, to run with the file included, Davis_E_orig.WAV, and output to Davis_E_processed.WAV use:
381-project-phase-1 Davis_E_orig.WAV Davis_E_processed.WAV

The Summary.txt file will be placed in the current directory.
